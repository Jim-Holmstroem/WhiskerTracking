== WhiskerTracker Bachelordegree project in machinelearning. The goal is to use statistical methods to get reliable unsupervised rat whisker tracking in highspeed video.

== Project requirements: === WhiskerTracker

  • python >=2.6.5 http://www.python.org
  • numpy >=1.3.0 http://www.numpy.org
  • pygtk
  • pycairo

=== DataGenerator

  • ode
  • cairo
